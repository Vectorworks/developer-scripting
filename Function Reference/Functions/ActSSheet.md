# ActSSheet

## Description
Function ActSSheet returns the handle to the currently active worksheet.

```pascal
FUNCTION ActSSheet : HANDLE;
```

```python
def vs.ActSSheet():
    return HANDLE
```

## Remarks
OBSOLETE for Version 9: see new [GetTopVisibleWS](GetTopVisibleWS.md).
