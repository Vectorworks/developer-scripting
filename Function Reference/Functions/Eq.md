# Eq

## Description
Returns TRUE if the two real numbers are equal within the tolerance.

```pascal
FUNCTION Eq(
				value1    : REAL;
				value2    : REAL;
				tolerance : REAL): BOOLEAN;
```

    return BOOLEAN
