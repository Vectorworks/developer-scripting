# SheetList

## Description
Returns the name of the saved view specified by index.

```pascal
FUNCTION SheetList(sheetIndex : INTEGER): STRING;
```

```python
def vs.SheetList(sheetIndex):
    return STRING
```

## Parameters
|Name|Type|Description|
|---|---|---|
|sheetIndex|INTEGER|Index of the sheet|
