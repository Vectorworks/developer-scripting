# EXL_GetSheetIndex

## Description
Gets index of the Excel sheet.

```pascal
FUNCTION EXL_GetSheetIndex(
				sheetName         : STRING;
				VAR outSheetIndex : INTEGER): BOOLEAN;
```

```python
def vs.EXL_GetSheetIndex(sheetName):
    return (BOOLEAN, outSheetIndex)
```
