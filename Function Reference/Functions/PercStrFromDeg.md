# PercStrFromDeg

## Description
Converts slope from degrees to percent string.

```pascal
FUNCTION PercStrFromDeg(fSlopeDeg : REAL): STRING;
```

```python
def vs.PercStrFromDeg(fSlopeDeg):
    return STRING
```
