# IFC_GetDSCount

## Description
Gets Data Sheets count for object.

```pascal
FUNCTION IFC_GetDSCount(
				objectName     : STRING;
				VAR outDSCount : INTEGER): BOOLEAN;
```

```python
def vs.IFC_GetDSCount(objectName):
    return (BOOLEAN, outDSCount)
```
