# LightingInvImport

## Description
Imports Lighting inventory from data exchange file. Returns TRUE if items were imported.

```pascal
FUNCTION LightingInvImport : BOOLEAN;
```

```python
def vs.LightingInvImport():
    return BOOLEAN
```
