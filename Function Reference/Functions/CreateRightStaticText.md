# CreateRightStaticText

## Description
Similar to CreateStaticText, but creates static text that is right-justified in its control field on the dialog.

```pascal
PROCEDURE CreateRightStaticText(
				dialogID          : LONGINT;
				itemID            : LONGINT;
				text              : STRING;
				widthInCharacters : INTEGER);
```
