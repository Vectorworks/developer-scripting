# InsertImagePopupSeparator

## Description
Inserts a separator with the specified label at the end of the image popup list.

```pascal
FUNCTION InsertImagePopupSeparator(
				liDialogID    : LONGINT;
				liComponentID : LONGINT;
				strLabel      : STRING): INTEGER;
```
