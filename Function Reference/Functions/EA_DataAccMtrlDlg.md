# EA_DataAccMtrlDlg

## Description
Shows the lambda material dialog and set the selected lambda value to outLambda.

```pascal
FUNCTION EA_DataAccMtrlDlg(
				acc           : INTEGER;
				VAR outLambda : REAL): BOOLEAN;
```

```python
def vs.EA_DataAccMtrlDlg(acc):
    return (BOOLEAN, outLambda)
```
