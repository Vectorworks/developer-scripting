# RRectangleN

## Description
Creates and returns a handle to a new rotated rectangle object with the specified bounds

```pascal
PROCEDURE RRectangleN(
				orginX,orginY         : REAL;
				directionX,directionY : REAL;
				width                 : REAL;
				height                : REAL;
				xDiam                 : REAL;
				yDiam                 : REAL);
```
