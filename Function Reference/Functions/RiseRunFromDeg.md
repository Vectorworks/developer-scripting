# RiseRunFromDeg

## Description
Converts slope from degrees to rise-over-run string.

```pascal
FUNCTION RiseRunFromDeg(fSlopeDeg : REAL): STRING;
```

```python
def vs.RiseRunFromDeg(fSlopeDeg):
    return STRING
```
