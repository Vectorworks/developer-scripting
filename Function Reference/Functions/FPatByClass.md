# FPatByClass

## Description
Sets the default document fill pattern to the fill pattern of the active class.

```pascal
PROCEDURE FPatByClass;
```

```python
def vs.FPatByClass():
    return None
```

## Remarks
Sets the global attributes to use the fill pattern of the default class.
