# DegFromStr

## Description
Converts slope from string format to degrees.

```pascal
FUNCTION DegFromStr(
				fSlopeDef   : STRING;
				fSlopeValue : STRING;
				VAR fAngle  : REAL): BOOLEAN;
```

```python
def vs.DegFromStr(fSlopeDef, fSlopeValue):
    return (BOOLEAN, fAngle)
```
