# DLDSetLoadDataReal

## Description
Using selector, sets default load data with real value for the parametric object.
Available selectors : kDLDSelectorWeight = 5.

```pascal
PROCEDURE DLDSetLoadDataReal(
				selector : INTEGER;
				value    : REAL);
```
