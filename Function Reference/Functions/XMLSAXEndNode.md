# XMLSAXEndNode

## Description
Write XML using SAX, end of a node. [ XMLSAXBeginNode](XMLSAXBeginNode.md) begins a node.

```pascal
FUNCTION XMLSAXEndNode(XMLHandle : LONGINT): INTEGER;
```
