# IFC_ImportWithUI

## Description
Imports IFC file using the IFC Import dialog.

```pascal
PROCEDURE IFC_ImportWithUI;
```

```python
def vs.IFC_ImportWithUI():
    return None
```

## Examples
#### VectorScript ####
```pascal
PROCEDURE Test;

BEGIN
	ok := IFC_ImportWithUI();
END;

RUN(Test);
```
#### Python ####
```python
ok = vs.IFC_ImportWithUI()
```

Availability: from Vectorworks 2014
