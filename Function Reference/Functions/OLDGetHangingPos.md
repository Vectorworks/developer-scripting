# OLDGetHangingPos

## Description
Returns handle of the Hanging Position where the specified load is attached.

```pascal
FUNCTION OLDGetHangingPos(
				handle    : HANDLE;
				loadIndex : INTEGER): HANDLE;
```

```python
def vs.OLDGetHangingPos(handle, loadIndex):
    return HANDLE
```
