# WSScript_GetObject

## Description
This function must be used inside a worksheet script called by 'RunScript' worksheet formula.

```pascal
FUNCTION WSScript_GetObject : HANDLE;
```

```python
def vs.WSScript_GetObject():
    return HANDLE
```
