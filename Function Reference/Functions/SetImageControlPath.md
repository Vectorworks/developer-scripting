# SetImageControlPath

## Description
Sets the image control path for the specified layout manager image control.  Use with CreateImageControl.

```pascal
FUNCTION SetImageControlPath(
				nDialogID    : LONGINT;
				nComponentID : LONGINT;
				strPath      : STRING): BOOLEAN;
```
