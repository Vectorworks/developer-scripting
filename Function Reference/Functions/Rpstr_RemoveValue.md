# Rpstr_RemoveValue

## Description
Removes a named value from the VectorScript value repository.

```pascal
FUNCTION Rpstr_RemoveValue(name : STRING): BOOLEAN;
```

```python
def vs.Rpstr_RemoveValue(name):
    return BOOLEAN
```

## Parameters
|Name|Type|Description|
|---|---|---|
|name|STRING|The name of the value.|
