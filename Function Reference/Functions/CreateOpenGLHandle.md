# CreateOpenGLHandle

## Description
Return a new OpenGL options handle.

```pascal
PROCEDURE CreateOpenGLHandle(VAR GLHandle : HANDLE);
```

```python
def vs.CreateOpenGLHandle():
    return GLHandle
```
