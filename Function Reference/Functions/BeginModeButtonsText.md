# BeginModeButtonsText

## Description
Creates a mode bar buttons help text.

```pascal
PROCEDURE BeginModeButtonsText;
```

```python
def vs.BeginModeButtonsText():
    return None
```

## Examples
```pascal
BeginModeButtonsText;
SetModeButtonText( 'Mode1', 1 );
SetModeButtonText( 'Mode2', 0 );
EndModeButtonsText;
```
