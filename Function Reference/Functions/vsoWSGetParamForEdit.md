# vsoWSGetParamForEdit

## Description
Get the parameter that a worksheet is trying to edit.

```pascal
PROCEDURE vsoWSGetParamForEdit(VAR paramName : STRING);
```

```python
def vs.vsoWSGetParamForEdit():
    return paramName
```
