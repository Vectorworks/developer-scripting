# EXL_GetSheetSize

## Description
Get size of Excel sheets.

```pascal
FUNCTION EXL_GetSheetSize(
				sheetIndex     : INTEGER;
				VAR outRows    : INTEGER;
				VAR outColumns : INTEGER): BOOLEAN;
```

```python
def vs.EXL_GetSheetSize(sheetIndex):
    return (BOOLEAN, outRows, outColumns)
```
