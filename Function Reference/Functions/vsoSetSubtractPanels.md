# vsoSetSubtractPanels

## Description
Used during event 58 to retrun whether a curtain wall object will have the wall subtract the panel from the frames

```pascal
PROCEDURE vsoSetSubtractPanels(inSubtractPanels : BOOLEAN);
```

```python

    return None
```
