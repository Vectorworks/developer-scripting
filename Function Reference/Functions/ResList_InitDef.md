# ResList_InitDef

## Description
Initialize a categories resource with red symbol resources of the specified universal name. The 'uniqueID' is a string identifier uniquely identifying this control.

```pascal
PROCEDURE ResList_InitDef(
				uniqueID : STRING;
				univName : STRING);
```

    return None
