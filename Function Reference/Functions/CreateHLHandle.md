# CreateHLHandle

## Description
Create a new Hidden Line Rendering options handle.

```pascal
PROCEDURE CreateHLHandle(VAR HLOptionsHandle : HANDLE);
```

```python
def vs.CreateHLHandle():
    return HLOptionsHandle
```
