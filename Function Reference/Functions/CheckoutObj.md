# CheckoutObj

## Description
Checkouts all objects which match the search criteria.

```pascal
PROCEDURE CheckoutObj(c : CRITERIA);
```

```python
def vs.CheckoutObj(c):
    return None
```

## Parameters
|Name|Type|Description|
|---|---|---|
|c|CRITERIA|Search criteria|

## Version
Availability: from Vectorworks 2017
