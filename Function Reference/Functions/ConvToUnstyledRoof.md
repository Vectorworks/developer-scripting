# ConvToUnstyledRoof

## Description
Sets a roof to be unstyled.

```pascal
PROCEDURE ConvToUnstyledRoof(roof : HANDLE);
```

```python
def vs.ConvToUnstyledRoof(roof):
    return None
```

## Parameters
|Name|Type|Description|
|---|---|---|
|roof|HANDLE|The roof.|
