# GetGISOrigin

## Description
Get geographical origin.

```pascal
FUNCTION GetGISOrigin(
				VAR outLat          : REAL;
				VAR outLon          : REAL;
				VAR outAngleToNorth : REAL): BOOLEAN;
```
