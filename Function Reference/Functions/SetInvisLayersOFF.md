# SetInvisLayersOFF

## Description
Sets the invisible exported DXF layers to OFF

```pascal
PROCEDURE SetInvisLayersOFF(invisLayerOFF : BOOLEAN);
```

```python

def vs.SetInvisLayersOFF(invisLayerOFF):
    return None
```
