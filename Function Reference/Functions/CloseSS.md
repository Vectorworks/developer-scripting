# CloseSS

## Description
Procedure CloseSS closes the referenced worksheet.

```pascal
PROCEDURE CloseSS(h : HANDLE);
```

```python
def vs.CloseSS(h):
    return None
```

## Parameters
|Name|Type|Description|
|---|---|---|
|h|HANDLE|Handle to worksheet.|
