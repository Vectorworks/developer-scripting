# IFC_DMIsEntryEnabled

## Description
Checks if the indicated entry from current IFC Data Mapping is enabled.

```pascal
FUNCTION IFC_DMIsEntryEnabled(
				inStrObjName   : STRING;
				inStrEntryName : STRING): BOOLEAN;
```
