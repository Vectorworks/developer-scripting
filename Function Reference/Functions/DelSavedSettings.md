# DelSavedSettings

## Description
Delete all saved settings in the specified category.

```pascal
FUNCTION DelSavedSettings(category : STRING): BOOLEAN;
```

```python
def vs.DelSavedSettings(category):
    return BOOLEAN
```

## Parameters
|Name|Type|Description|
|---|---|---|
|category|STRING|The category to be deleted.|
