# vsoStyleWidgetChoice

## Description
Get the item chosen in the Plug-in Object Style widget.

```pascal
PROCEDURE vsoStyleWidgetChoice(VAR choice : INTEGER);
```

```python
def vs.vsoStyleWidgetChoice():
    return choice
```
