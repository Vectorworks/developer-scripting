# EXL_GetSheetName

## Description
Get name of Excel sheets.

```pascal
FUNCTION EXL_GetSheetName(
				sheetIndex       : INTEGER;
				VAR outSheetName : STRING): BOOLEAN;
```

```python
def vs.EXL_GetSheetName(sheetIndex):
    return (BOOLEAN, outSheetName)
```
