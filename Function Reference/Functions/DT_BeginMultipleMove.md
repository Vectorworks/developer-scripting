# DT_BeginMultipleMove

## Description
Returns TRUE if moving system was started successful.

```pascal
FUNCTION DT_BeginMultipleMove : BOOLEAN;
```

```python
def vs.DT_BeginMultipleMove():
    return BOOLEAN
```
