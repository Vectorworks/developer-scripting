# ShowCreateImageDialog

## Description
Displays a dialog that allows the user to select an image file from which a new image resource is created.

```pascal
FUNCTION ShowCreateImageDialog : HANDLE;
```

```python
def vs.ShowCreateImageDialog():
    return HANDLE
```
