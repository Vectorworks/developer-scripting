# FillColorByClass

## Description
Sets the default document fill colors to the fill colors of the active class.

```pascal
PROCEDURE FillColorByClass;
```

```python
def vs.FillColorByClass():
    return None
```

## Remarks
Sets the global attributes to use the fill colors of the default class.
