# IFC_ExportNoUI

## Description
Exports IFC file, without showing Export IFC Project dialog.

```pascal
FUNCTION IFC_ExportNoUI(strFullFilePath : DYNARRAY[] of CHAR): BOOLEAN;
```

```python
def vs.IFC_ExportNoUI(strFullFilePath):
    return BOOLEAN
```
