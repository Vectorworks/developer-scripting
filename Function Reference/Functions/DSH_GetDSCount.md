# DSH_GetDSCount

## Description
Gets Data Sheets count for Object.

```pascal
FUNCTION DSH_GetDSCount(
				hObject      : HANDLE;
				VAR outCount : INTEGER): BOOLEAN;
```

```python
def vs.DSH_GetDSCount(hObject):
    return (BOOLEAN, outCount)
```
