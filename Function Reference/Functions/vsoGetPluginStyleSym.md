# vsoGetPluginStyleSym

## Description
Allow an object to use its own implementation in creating and maintaining a plugin style

```pascal
PROCEDURE vsoGetPluginStyleSym(VAR hSymDef : HANDLE);
```

```python
def vs.vsoGetPluginStyleSym():
    return hSymDef
```
