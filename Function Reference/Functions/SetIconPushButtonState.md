# SetIconPushButtonState

## Description
Retrieves the state of the specified Layout Manager icon push button (pressed or not pressed).

```pascal
FUNCTION SetIconPushButtonState(
				nDialogID    : LONGINT;
				nComponentID : LONGINT;
				bPressed     : BOOLEAN): BOOLEAN;
```
