# ClassNum

## Description
Returns the total number of classes in the active document.

```pascal
FUNCTION ClassNum : LONGINT;
```

```python
def vs.ClassNum():
    return LONGINT
```

## Examples
#### VectorScript ####
```pascal
numOfClasses:= ClassNum;
```
#### Python ####
```python
numOfClasses = vs.ClassNum()
```

Availability: from All Versions
